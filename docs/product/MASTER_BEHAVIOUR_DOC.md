

GOAL:
Define system behavior for SEO platform.

---

PRIMARY FLOW:

INPUT → ANALYSIS → INSIGHT → PRIORITY → ACTION

---

INPUT TYPES:

- website URL
- app URL
- keywords
- reviews
- competitor URLs/apps

---

OUTPUT TYPES:

- keyword insights
- ranking insights
- content insights
- review insights
- competitor insights
- actions

---

BEHAVIOUR RULES:

RULE 1:
Convert data → insight

RULE 2:
Convert insight → action

RULE 3:
Prioritize all outputs

RULE 4:
Avoid overload

RULE 5:
Highlight highest impact item

---

STATE TYPES:

- loading
- success
- empty
- error

---

CONSTRAINT:

System must not output raw data without interpretation
